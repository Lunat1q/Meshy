# Meshy
Small demo of mesh animation in JS


## Hotkeys

### General
- `Spacebar` = add new point
- `Enter` = pop point

### Rendering settings

- `L` = toggle lines
- `T` = toggle triangles
- `P` = toggle points

### Animation:

- `A` = toggle animation
- `{ (Left bracket)` = increase trail time animation
- `} (Left bracket)` = decrease trail time animation
- `+` = increase points speed
- `-` = decrease points speed
